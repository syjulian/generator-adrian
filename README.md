## Installation

clone the project from github
```bash
git clone https://github.com/syjulian/generator-adrian.git
```

go into the project and 'install'
```bash
cd generator-adrian

# "install" to node
npm link
```

## Getting Started
Create your project folder
```bash
mkdir myProject

cd myProject
```

Initiate the generator
```bash
yo adrian
```
